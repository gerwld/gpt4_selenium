# Init project:

1. Clone repository
2. Run

```
pip: -r requirements.txt
```

# Tools:

### getlinks:

```
python -m tools.getlinks
```

### parseNews

```
python -m tools.parseNews
```

### convertToMdPosts

```
python -m tools.convertToMdPosts
```
